# 📰 Berita Kini - SEAL Frontend Technical Test

Proyek ini dikembangkan sebagai bagian dari **Technical Test Magang Mandiri SEAL (Social Economic Accelerator Lab) Batch 1 2026**. Aplikasi "Berita Kini" adalah portal berita responsif yang dikembangkan berdasarkan desain Figma yang telah ditentukan, dengan fitur pencarian dan kategori berita secara real-time.

---

## 🚀 Teknologi yang Digunakan

*   **Core Framework**: React.js (Vite)
*   **Styling**: Tailwind CSS v4 & CSS3 Custom Properties
*   **Routing**: React Router Dom v6
*   **State Management**: React Hooks (`useState`, `useEffect`)
*   **API Sources**: 
    *   CNN News API (Nasional, Olahraga, Internasional)
    *   CNBC News API (Teknologi)
*   **Font**: Plus Jakarta Sans (Google Fonts)

---

## ✨ Fitur Utama

*   **Responsive UI**: Tampilan optimal di perangkat mobile, tablet, maupun desktop.
*   **Headline Slider**: Navigasi pagination pada section berita utama (Hero).
*   **Real-time Search**: Fitur pencarian berita di section rekomendasi (Frontend Filtering).
*   **Multi-Source API**: Integrasi dengan API CNN dan CNBC Indonesia untuk menyajikan berita terkini.
*   **Dynamic News Detail**: Halaman detail berita yang lengkap dengan section komentar dan berita terkait.
